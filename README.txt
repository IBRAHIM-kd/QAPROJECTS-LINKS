HOW TO RUN THIS PROJECT - BELOW ARE THE STEPS AS FOLLOWS : -


Before we start Protractor, we need to install the following

Download and install
----------------------------------------------------------
Java JDK 8
NPM (Node.js) v10.16.0
Protractor/Selenium 
jasmine-reporters@^2.0.0 
----------------------------------------------------------

To Execution of the Code

Follows this STEP ---


Here first, we will change the directory path or navigate to the folder where the confi.js and spec.js are placed in our system.

Follow the following step.

Step 1) Open the command prompt on Windows OS or shell on linux OS.

Step 2) Make sure selenium web driver manager is up and running. For that give the command as "webdriver-manager start" and hit Enter.

Step 3) Open a new command prompt and give the command as "protractor conf.js" to run the configuration file.


Thats All.....!
